Space Tower Defence

Space Tower Defence was our team's final project for CS1420 (Accelerated Object Oriented Programming) at the University of Utah. This game is a space version of balloon tower defence, where there are comets that are released, and 2 types of towers that can be placed once enough points have been gained. The game ends once the user loses all their hitpoints, which happens when the comets keep attacking the base for an extended amount of time.
